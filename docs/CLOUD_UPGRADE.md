# CiteOps Cloud Upgrade Path

CiteKit is designed to create an immediate local value moment:

1. Run `citekit doctor`.
2. Run a benchmark or scan.
3. See concrete crawl/citation gaps.
4. Export or share JSON evidence.
5. Upgrade to CiteOps Cloud when the user wants autonomous execution.

## CiteKit OSS Gives

- Local CLI scans and benchmarks.
- Provider readiness checks.
- Diagnosis and fix recommendations.
- JSON outputs for engineering handoff.

## CiteOps Cloud Adds

- Scheduled monitoring.
- Competitor share-of-answer tracking.
- Website platform detection and connector routing.
- GitHub PRs and CMS drafts.
- Human approval gates.
- Verification loops.
- Public proof cards and scoreboards.
- TraceIntelligence audit trails.
- Hashed-only Sansa learning.
